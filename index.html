ScreenpLaTeX

A markdown language for screenwriting.

ScreenpLaTeX Commands:

ACTION - \a{some text} - For actions/scene directions. (e.g. \a{EARN races MICHAEL VICK.})


TRANSITION - \t{some text} - For transitions. (e.g. \t{cut to:})


SUBHEAD - \subhead{some text} - For subheadings, changing setting within a set. (e.g. \subhead{uber})


SLUGLINE - \slugline(int or ext)(time of day){some setting} - For setting heading, or "slugline". (e.g. \slugline(int)(night){TEDDY PERKINS'S Mansion.})


CHARACTER - \c(character name){some dialogue} - For character dialogue. (e.g. \c(Darius){Smoke me!})


CHARACTER PARENTHETICAL - \cp(character name)(character action){some dialogue} - For character directions during dialogue. (e.g. \cp(Al)(opening wing box){Oh, fuck.})


CHARACTER EXTENSION - \ce(character name)(stage instructions){some dialogue} - For character dialogue with specific constraints (voice over, character off-screen, etc.) (e.g. \ce(Clark County)(O.S.){Yoo hoo!})


CHARACTER EXTENSION PARANTHETICAL - \cep(character name)(stage instructions)(character action){some text} - For character dialogue with specific constraints AND character directions during dialogue. (e.g. \cep(Al)(v.o.)(sarcastically){I thought drugs were for loooooosers.})
